Docker

Docker is a tool designed to make it easier to create, deploy, and run applications by using containers. Containers allow a developer to package up an application with all of the parts it needs, such as libraries and other dependencies, and ship it all out as one package.

Docker Commands

Description Command
Docker images which are pulled from hub.docker.com docker pull image_name:version
To see the list of running containers docker ps
To see the list of all containers docker ps -a
To remove a image docker rmi image_name
To remove a container docker rm container_name
To execute a command in the container docker exec container_name command
To run processes in isolated containers docker run options image
To start docker services systemctl start docker
To see the docker status systemctl status docker
